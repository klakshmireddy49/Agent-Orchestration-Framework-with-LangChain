
# Milestone 3: Multi-Agent Orchestration & Memory Management

## Objective
Enable collaboration between multiple agents and implement memory-based reasoning to improve decision-making.

## Tasks
- Define multi-agent roles (e.g., Research Agent, Summarizer Agent)  
- Implement communication between agents:
  - Chained calls  
  - Shared scratchpad  
- Add individual memory to agents (e.g., ConversationBufferMemory)  
- Implement shared memory (e.g., VectorStoreRetrieverMemory)  
- Orchestrate a complete scenario involving agent collaboration  
- Ensure memory updates influence future decisions and responses  

## Output
- Multi-agent system with clearly defined agent roles  
- Functional communication and memory layers across agents  
- Successful collaborative task execution influenced by memory
