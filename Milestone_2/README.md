# Milestone 2: Tool Integration & API Calling

## Objective
Extend the agent’s functionality by integrating custom tools and enabling API-based interactions.

## Tasks
- Understand LangChain’s Tool abstraction  
- Implement at least two tools:
  - Calculator  
  - Simulated Weather API  
- Integrate tools into the agent’s workflow  
- Design prompts that guide the agent to use tools appropriately  
- Test tool invocation and response handling  
- Implement error handling for API failures  

## Output
- LangChain agent with tool usage capability  
- At least two integrated tools (mock APIs)  
- Demonstration of agent interacting with tools effectively
